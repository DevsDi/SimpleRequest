SimpleRequest Privacy Policy

Last Updated: June 4, 2024

Overview

SimpleRequest is a lightweight HTTP request debugging tool. We value your privacy and this policy explains how we collect, use, and protect your information.

Data Collection

Data We Collect

Data TypeStorage LocationPurposeRetention
Request historyLocal browser storage (chrome.storage.local)Quick re-send requestsManual clear or max 100 entries
User preferencesLocal browser storageRemember layout settingsPersistent

Data We Do NOT Collect

  • ❌ No personal identity information
  • ❌ No browsing history
  • ❌ No cookies or other website data
  • ❌ No data sent to any server
  • ❌ No analytics or tracking tools

Data Storage

All data is stored locally in your browser (chrome.storage.local). No data is uploaded to any server. Data exists only on your device — we cannot access it.

Storage Details

  • History: Max 100 entries, single response max 2MB
  • Preferences: Window layout, panel widths, etc.
  • Clear method: Click “Clear” button to delete all history

Data Usage

Collected data is used only for:

  1. History: Quick re-send previous requests
  2. Layout preferences: Restore your adjusted UI on next open

Third Party Access

We do not share any data with third parties. Data is stored entirely locally and cannot be accessed externally.

Data Security

  • Data stored in Chrome's local storage, protected by browser security
  • Requests sent via Chrome extension APIs, not through third-party servers
  • You can clear all data anytime

Your Rights

You have the following rights:

  • View data: Open extension to see all history
  • Clear data: Click “Clear” button to delete history
  • Uninstall: All local data deleted upon uninstall

Cookie Usage

This extension does not use cookies or any tracking technologies.

Children's Privacy

This extension is not intended for children under 13. We do not knowingly collect children's personal information.

Policy Updates

If this policy changes significantly, we will notify you in extension update notes. Review this policy regularly.
